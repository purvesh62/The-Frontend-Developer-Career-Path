The CSS Box Model

We can declare widths and heights on elements to define their size. We also have other properties like margins, padding, and borders which make up the box model.

Margin

Margin is used to control the space around an element.

Padding

Padding is a lot like margin, except that it adds the empty space inside the element, which includes it's background-color.

Borders

Borders are a line that surrounds an element, similar to a stroke if you are familiar with design software.
